Maps
======

Создайте проект Maps-task, где в отдельных
классах выполните задания.


Задание 1
------------
1) При помощи HashMap создайте
перечень элементов заказа:
товар (ключ), количество (значение).
2) В одном из товаров изменено количество.
3) Добавлен новый товар.
4) Реализуйте: вывод начального перечня
товаров, обновление товара, добавление товара,
вывод последующего перечня товаров.
5) Отдельные действия/задачи реализуйте
через отдельные методы.


Задание 2
------------
1) При помощи HashMap создайте
перечень товаров:
наименование (ключ), количество (значение).
2) Средствами Scanner реализуйте
поиск количества по наименованию.
3) Формирование перечня, ввод поиска
и вывод результата сделайте каждый в
отдельном методе.


Задание 3
------------
1) При помощи LinkedHashMap создайте
перечень контактов:
имя (ключ), э-почта (значение).
2) Один из контактов сменил э-почту.
3) Реализуйте: вывод начального перечня
контактов, смену э-почты, вывод
последующего перечня контактов.
4) Отдельные действия/задачи реализуйте
через отдельные методы.
